# Program Resource Bucket Rubric (V1)

Score each item 0/1. Total 0–5.

1) Dedicated strength program (coach-led) available year-round
2) Team-provided film / analytics platform access (Hudl or equivalent)
3) Offseason training schedule >= 3 days/week organized by staff
4) Access to quality facilities (weight room + field time without restrictions)
5) Staff depth: >= 2 position coaches or documented specialist support

Bucket:
- 0–1 = low
- 2–3 = medium
- 4–5 = high

Note: This measures program resources, not family resources.
